    Exercises 1.1.3 Exercises

    1.

    Set a variable called "number" equal to 5. Add 10 to "number", multiply "number" by 5, and subtract 7. Finally, print number. What is the value of number?
    Solution.
    Remember at each stage, you must reassign the value to number. This is done with the equals sign. Finally, print() will print the variable you pass to it. NOtice that the operations do not have to be on one line. Python will follow order of operations, so you can multiply by 5 and subtract 7 on the same line. 
    number = 5
number = number + 10
number = 5*number - 7
print(number)
